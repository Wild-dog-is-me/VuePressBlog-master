## 前言

后端狗，不怎么关心前端的事情，由于特殊原因，需要切换到其他node版本。经朋友推荐，有一个很好的解决方案，选择node版本管理工具，轻松切换node。

## 安装

```bash
$ npm install -g n
```

下载完就可以使用```n```了

## 利用n下载所需要的node

```bash
$ n 版本号   
```

下载最新版本

```bash
$ n latest
```

删除某个版本

```bash
$ n rm 4.4.4 
```

查看当前node版本

```bash
$ node -v
```

切换版本

```bash
$ n

  6.9.4
ο 7.4.0
  4.4.4
```

查看版本

```bash
$ node -v
```
